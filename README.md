# Todo App

シンプルで使いやすいTodoアプリケーション

## 機能

- **CRUD操作**: タスクの追加、表示、完了、削除
- **優先度設定**: 高・中・低の3段階で優先度を設定
- **フィルタリング**: すべて/未完了/完了でタスクをフィルター表示
- **データ永続化**: localStorageを使用してデータを保存

## 使い方

1. ブラウザで `index.html` を開く
2. 入力欄に新しいタスクを入力
3. 優先度を選択（デフォルト: 中）
4. 「追加」ボタンをクリック、またはEnterキーで追加
5. チェックボックスでタスクを完了/未完了に切り替え
6. 削除ボタンでタスクを削除

## 技術スタック

- HTML5
- CSS3
- Vanilla JavaScript (ES6+)
- localStorage API

## ファイル構成

- `index.html` - アプリケーションのメインHTML
- `styles.css` - スタイルシート
- `app.js` - アプリケーションロジック
